# Quickport ([Nexus Mods Mirror](https://www.nexusmods.com/skyrimspecialedition/mods/68063?tab=description))

A small mod created as part of a friends request, adding a lesser power to teleport to a random location within Skyrim's Worldspace.

Optionally integrating into [Alternate Perspective](https://github.com/KrisV-777/Alternate-Perspective).
